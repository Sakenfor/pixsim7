# Scene Panels

Panels related to scene management, editing, and playback.

## Panels in this directory

- **SceneBuilderPanel** - Build and edit individual scenes
- **SceneCollectionPanel** - Manage scene collections
- **SceneLibraryPanel** - Browse and manage scene library
- **SceneManagementPanel** - Unified scene workflow management
- **ScenePlaybackPanel** - Scene playback controls and preview

## Usage

These panels are registered in the core panels plugin and provide the main UI for working with scenes throughout the application.

# Shared Panel Infrastructure

Shared panel building blocks and infrastructure components.

## Components in this directory

- **ComposedPanel** - Panel composition system for widget-based panels
- **SimplePanelBuilder** - Simplified UI for creating composed panels
- **FloatingPanelsManager** - Manager for floating/draggable panels

## Usage

These are infrastructure components used by other panels and features:
- `ComposedPanel` is used by HUD system and panel builder
- `SimplePanelBuilder` provides UI for composing panels from widgets
- `FloatingPanelsManager` handles all floating panel instances

Import these components when building custom panels or panel-based features.

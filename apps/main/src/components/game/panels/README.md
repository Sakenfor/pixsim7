# Game Panels

Panels related to game functionality, theming, and player interaction.

## Panels in this directory

- **DynamicThemeRulesPanel** - Dynamic theme rules editor (Game World → World Tools → Theme Rules)
- **HudCustomizationPanel** - HUD layout customization
- **InteractionPresetUsagePanel** - Interaction preset usage and stats
- **InventoryPanel** - Player inventory management
- **NpcInteractionPanel** - NPC interaction interface
- **SessionOverridePanel** - Session theme override (Game 2D → Session Theme Override world tool)
- **ThemePacksPanel** - Theme pack management (Game World → World Tools → Theme Packs)
- **UserPreferencesPanel** - Accessibility & UI preferences (Settings → Appearance → Accessibility)
- **WorldToolsPanel** - World management tools

> The former **GameThemingPanel** (a four-tab container) was dissolved; its tabs
> now live in their natural scopes as noted above.

## Usage

These panels provide game-specific functionality and are used in game routes, world management, and player-facing features.
